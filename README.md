### 🛠 Сборка и запуск

Требования: 
- Qt 6.9 и выше
- Git (для клонирования репозитория)
- Убедитесь, что у вас установлен дополнительный компонент QtCharts
  
Запуск программы: 
1. Откройте папку для проекта в терминале и клонируйте репозиторий с помощью следующей команды:
```text
git clone https://github.com/vazzorina/Genetic-Algorithm
```
2. Через Qt Creator откройте файл `CMakeLists.txt`. Выберите нужный компилятор и конфигурацию.
3. Нажмите **Build** - для сборки проекта
4. Нажмите **Run** - для запуска проекта

### Описание

Проект разработан в целях изучения и практического применения генетического алгоритма.

Целью генетического алгоритма является нахождение **минимума целевой функции**:

$$y = 2x - 3.3x \cdot \cos(9.3x) - 2.3 \cdot \sin(1.7x)$$

При этом установлены следующие ограничения:
- Количество особей в популяции: `50`
- Область значений х: `[-5, 5]`
- Длина хромосомы: `20 бит`
- Кодирование хромосом: `двоичный код`
- Вероятность скрещивания: `0.9`
- Вероятность мутации: `0.2`
- Формирование родительских пар: `случайный отбор`
- Редукция: `турнир`
- Количество циклов генетического алгоритма: `50`

### Реализованные функции:
- Получение значения целевой функции
- Получение значения функции приспособленности
- Выделение пар родителей случайным отбором
- Генерация потомков и их мутация
- Турнир между всеми особями популяции
- Запуск цикла генетического алгоритма
- Получение среднего значения функции приспособленности
- Запись данных особей в файл csv

> **Получение хромосомы**
>
> Так как `x` может быть отрицательным дробным числом, просто разместить его в `std::bitset<20>` будет неверным решением, так как кодирование дробных и целых чисел со знаком и без знака различаются (!). Поэтому я пришла к следующему решению:
>```text
>int X = (x + 5) * 100000;
>std::bitset<20> chrom(X);
>```
>
> Я прибавляю к `х` число 5 для того, чтобы избавиться от знака, и умножаю на 100000, чтобы сохранить дробную часть. Умножаю именно на 100000, так как максимальное десятичное число, которое может быть закодировано 20 битами - это 1048575. Максимальным `x` может быть 10 и плюс дробная часть 5 знаков.
>
